## Local Development

1. Clone this repository:
```bash
git clone https://github.com/YOUR_USERNAME/twitch-dashboard.git
cd twitch-dashboard
```

2. Open `twitch_dashboard.html` in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

3. Navigate to `http://localhost:8000` (or the appropriate port)

## Customization

- **Color Scheme**: CSS variables in the `<style>` section
- **Timeline Data**: Lines 462-486 for weekly breakdown
